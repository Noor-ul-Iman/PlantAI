About the Dataset

This dataset is recreated using offline augmentation from the original dataset. The original dataset can be found on this github repo. This dataset consists of about 87K rgb images of healthy and diseased crop leaves which is categorized into 38 different classes. The total dataset is divided into 80/20 ratio of training and validation set preserving the directory structure.

The images are split into training, validation, and test sets:

  • Train: 70,295 images
  • Validation: 17,572 images
  • Test: 33 images

This dataset helps train deep learning models for plant disease classification. The original version can be found on GitHub, and this augmented set follows the same directory structure with an 80/20 split.
